

# XELA - Your AI Assistant (1.O the first edition)


## Overview

XELA (pronounced like "Zay-la") is your personal AI assistant built to assist you with various tasks and answer questions using OpenAI's GPT-3.5 Turbo model. Whether you want to open applications, search the web, get the current time, or engage in a chat, XELA is here to help.

## Features

- **Voice Commands:** XELA can understand your voice commands and respond accordingly.
- **Web Browsing:** Ask XELA to open websites like Google, Facebook, or YouTube.
- **Application Launch:** Open applications such as Terminal, Chrome, or Spotify with ease.
- **Web Search:** Simply ask XELA to search the web, and it will open a Google search for you.
- **AI Chat:** Engage in conversations with XELA powered by OpenAI's GPT-3.5 Turbo.
- **Time Display:** Ask XELA for the current time, and it will provide you with the time.

## Getting Started

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/XELA.git
   cd XELA
   ```

2. Install the required Python packages:

   ```bash
   pip install -r requirements.txt
   ```

3. Set up your OpenAI API key by replacing `YOUR_API_KEY` in the `config.py` file.

4. Run XELA:

   ```bash
   python XELA.py
   ```

## Usage

- Say "Hello XELA" to start a conversation.
- Ask XELA to perform tasks or answer questions using natural language.
- Use voice commands to open websites or applications.

## Customization

Feel free to customize XELA to suit your needs. You can extend its functionality by adding new voice commands, tasks, or integrations with other APIs.
